# SimplePost System

A simple post system for your projects. Written in JavaScript using Node.js, Express, MongoDB.


## Tech Stack

**Server:** Express, Mongoose, MongoDB, uuid, express-fileupload.



To launch the application use

```bash
  npm init
  and
  npm run start
```


## Dev

To build the project use

```bash
  npm init
  and
  npm run dev
```
